Lynx Security Demo

This interactive demo showcases three fundamental web security features implemented for the Lynx platform.

Security Features Demonstrated:

  • Content Security Policy (CSP) - Controls which scripts can execute
  • CSRF Protection - Prevents cross-site request forgery attacks
  • Input Sanitization - Cleans potentially dangerous HTML input

How to View This Demo

This demo requires the Lynx runtime to display properly. To view it:

  1. Make sure you have the Lynx Explorer app installed on your mobile device
  2. Open your terminal where you ran pnpm dev
  3. Scan the QR code displayed in the terminal with your mobile device's camera
  4. The demo will open automatically in the Lynx Explorer app

Note: For the best experience, view this demo on a mobile device using the Lynx Explorer app. The demo has been specifically optimized for the Lynx runtime environment with mobile-friendly UI elements.
