# create-rspeedy

Create a new Rspeedy x ReactLynx project.

Using `npm create`:

```bash
npm create rspeedy@latest
```

Using CLI flags:

```bash
npx create-rspeedy --dir my-project

# Using abbreviations
npx create-rspeedy -d my-project
```
